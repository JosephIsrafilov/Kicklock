# Recorded pitch fixtures

The two MP3 previews in this directory are derived from the following genuine
recordings. They deliberately do **not** use the `real_kick.wav` /
`real_bass.wav` names, so they cannot satisfy or alter the gated full-plugin
real-audio integration test.

| Fixture | Published note | Source | Licence |
| --- | --- | --- | --- |
| `cabled_mess_a1.wav` | A1, 55.0000 Hz | [Freesound #396122](https://freesound.org/people/cabled_mess/sounds/396122/) | CC0 1.0 |
| `cabled_mess_c2.wav` | C2, 65.4064 Hz | [Freesound #396121](https://freesound.org/people/cabled_mess/sounds/396121/) | CC0 1.0 |

The source describes a 4-string Squier Jazz Bass recorded through a Behringer
ADA8200, 48 kHz mono. The original WAV download requires a Freesound login;
the publicly served high-quality previews were decoded to 48 kHz mono WAV for
a small, reproducible regression asset. They exercise pitch classification
only, not the original-project kick/bass integration case.
